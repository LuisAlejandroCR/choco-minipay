# Agent Handoff

This repo is intentionally small and follows a clear public-app convention: root guidance docs, contract isolation, reusable components, and a centralized `src/lib` layer.

## Safe Commands

```bash
npm run check
npm run contracts:test
```

## Commit Style

Use small, block-oriented commits:

- `chore(structure): align repository architecture with reference implementation`
- `refactor(transfers): replace static recipient flow with Agent Choco AI detection`
- `feat(wallet): enforce verification-based access control`
- `fix(transactions): correct explorer links and QR payload generation`
